# Careers Website

## Overview
A modern, responsive careers website built with React and Vite. This website showcases open job positions in a clean, professional interface with department filtering and expandable job details.

## Project Structure
- **Frontend**: React 19 with Vite build tool
- **Port**: 5000 (configured for Replit environment)
- **Styling**: Custom CSS with responsive design

## Features
- Beautiful gradient header with company branding
- Department filtering sidebar
- Job listings with expandable details
- Responsive design for mobile and desktop
- Professional UI with smooth animations
- Sample job postings across Engineering, Design, and Marketing

## Technology Stack
- React 19.2.0
- Vite 7.2.2
- Modern JavaScript (ES6+)
- CSS3 with animations

## Development
The development server runs on port 5000 with hot module replacement (HMR) enabled. The Vite configuration is optimized for the Replit environment with proper host settings.

- Component-based React structure
